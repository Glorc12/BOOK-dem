# ЧитайГород — tkinter + PIL

Компактное решение для демонстрационного экзамена.

## Запуск

```bash
pip install -r requirements.txt
python scripts/build_db.py
python run.py
```

## Что внутри

- tkinter-интерфейс
- PIL для работы с изображениями
- SQLite база данных
- импорт данных из папки `import`
- роли: гость, клиент, менеджер, администратор
- каталог товаров, заказы, формы добавления и редактирования для администратора

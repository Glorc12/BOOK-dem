# Реализованные функции

1. Авторизация по логину и паролю.
2. Переход в режим гостя.
3. Просмотр каталога товаров.
4. Отображение фото, цены, скидки, остатка и описания.
5. Поиск, сортировка и фильтрация для менеджера и администратора.
6. Формы добавления и редактирования товара.
7. Формы добавления и редактирования заказа.
8. Удаление товара и заказа с проверками.
9. Работа с SQLite-базой.
10. Импорт справочников и исходных данных из папки `import`.
